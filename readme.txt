== Description ==

Writer Helper is a WordPress plugin design for helping authors to write, edit and publish books on their website.
For each book, you can track writing/editing progress of every scenes and chapters.

Features list:
* Create books, chapters, scenes
* Export a book in EPUB or text format
* Display the books anywhere on your website via shortcodes and widgets
* Create game book, book where you are the hero

== Installation ==

* Download the plugin from this page
* Navigate to **Plugins** > **Add New** > **Upload Plugin** page in admin panel of your website
* Click **Upload**, click **Choose file**, upload the plugin zip and click **Install Now** button
* Activate the plugin when installation is complete
